## About this project

This repository contains the solution to the **Self-Pruning Neural Network** case study,
submitted as part of the AI Engineer assessment.

The project implements a custom `PrunableLinear` layer with learnable gate parameters
and an **L1 sparsity loss**, enabling a neural network to dynamically prune its own
weights during training on CIFAR-10.
